# Subspace Product Teardown

This repository contains a product teardown of Subspace, prepared as part of a Product Intern assignment.

## Objective

Analyze the product through direct exploration, user feedback, competitor benchmarking, and product management frameworks to identify key growth opportunities.

## Key Findings

- Trust is the biggest barrier to adoption.
- Refund and transaction transparency need improvement.
- Marketplace reliability depends on stronger admin accountability.
- Product positioning and onboarding require greater clarity.
- Support visibility and feature discovery can improve engagement and retention.

## Frameworks Used

- SWOT Analysis
- Competitor Analysis
- Prioritization Framework
